# CORS Guide
This is the demo project for the blog post [A practical guide to CORS](https://xg-wang.github.io/2017/10/A-practical-guide-to-CORS/).

# Setup
```bash
git clone git@github.com:xg-wang/cors-guide.git
cd cors-guide
npm i
npm start
```
